Canvas & Frame State Variables ```javascript const scrollCanvas = document.getElementById('scrollCanvas'); const scrollCtx = scrollCanvas ? scrollCanvas.getContext('2d') : null; let scrollFrames = []; let currentFrame = 0; let isScrollAnimationEnabled = false; ``` **Purpose**: Track animation state and canvas context ### B. Canvas Sizing Function ```javascript function resizeScrollCanvas() { if (!scrollCanvas) return; scrollCanvas.width = window.innerWidth; scrollCanvas.height = window.innerHeight; } ``` **Purpose**: Responsive canvas resizing on window resize ### C. Main Loader: `loadScrollAnimation()` ```javascript async function loadScrollAnimation() { try { const frameCount = 89; // Typical output from 6s animation @ 15fps const framePromises = []; for (let i = 0; i < frameCount; i++) { const frameNum = String(i).padStart(4, '0'); const img = new Image(); img.src = `/sequence/frame_${frameNum}.webp`; framePromises.push( new Promise((resolve) => { img.onload = () => resolve(img); img.onerror = () => resolve(null); }) ); } scrollFrames = (await Promise.all(framePromises)).filter(f => f !== null); if (scrollFrames.length > 0) { isScrollAnimationEnabled = true; resizeScrollCanvas(); if (scrollCanvas) scrollCanvas.style.display = 'block'; drawScrollFrame(0); } } catch (err) { console.log('Scroll animation not available — WebP sequence not found'); } } ``` **Purpose**: Async loads WebP frame sequence from `/sequence/` directory **Customization**: - Change `frameCount` if your animation has different frame count - Change `frame_${frameNum}.webp` if your files have different naming ### D. Frame Rendering: `drawScrollFrame()` ```javascript function drawScrollFrame(frameIndex) { if (!scrollCtx || scrollFrames.length === 0) return; const clampedIndex = Math.max(0, Math.min(frameIndex, scrollFrames.length - 1)); const frame = scrollFrames[clampedIndex]; scrollCtx.clearRect(0, 0, scrollCanvas.width, scrollCanvas.height); scrollCtx.drawImage(frame, 0, 0, scrollCanvas.width, scrollCanvas.height); } ``` **Purpose**: Renders single frame to canvas at specified index ### E. Scroll Event Listener (Scroll-Linked Scrubbing) ```javascript window.addEventListener('scroll', () => { if (!isScrollAnimationEnabled) return; const scrollHeight = document.documentElement.scrollHeight - window.innerHeight; const scrollProgress = window.scrollY / (scrollHeight * 0.4); const frameIndex = Math.floor(scrollProgress * (scrollFrames.length - 1)); drawScrollFrame(frameIndex); }); ``` **Purpose**: Maps page scroll position to animation frame index **Logic**: - First 40% of page scroll = full animation play - Smooth interpolation between frame indices ### F. Resize Event Listener ```javascript window.addEventListener('resize', () => { if (isScrollAnimationEnabled) resizeScrollCanvas(); }); ``` **Purpose**: Keeps canvas responsive on viewport resize ### G. Activation Comment (Uncomment to Enable) ```javascript // Uncomment next line after setting up your WebP sequence: // loadScrollAnimation(); ``` **To activate**: Remove the `//` from this line --- ## 4.
